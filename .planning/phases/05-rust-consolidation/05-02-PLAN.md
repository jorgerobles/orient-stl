**1. `score_all_directions`** — per D-05 + D-01. Signature (locked with one extension — see below):

```rust
#[cfg(feature = "wasm")]
#[wasm_bindgen]
pub fn score_all_directions(
    positions: &[f32],
    normals: &[f32],
    areas: &[f32],
    directions: &[f32],
    critical_angle_deg: f32,
    refine_iters: u32,
    exclude_unstable: bool,
    progress: Option<&js_sys::Function>,
) -> Vec<f32>
```

NOTE on the locked spec: CONTEXT.md D-05 says the function returns "N×12 floats per direction [qx, qy, qz, qw, overhang, footprint, cross, surface, height, stable, margin, contact_area]". This EXCLUDES `shadowed`, which IS a locked metric per the phase goal ("Every scoring metric ... shadowed overhang ... has exactly ONE implementation"). The TS `rankByConsensus` uses shadowed as a 6th cost term — without shadowed in the output, consensus ranking cannot be reproduced in Rust.

**Resolution (planner discretion within D-05's intent): EXTEND the per-direction output to 13 floats by inserting `shadowed` AFTER `height`:** `[qx, qy, qz, qw, overhang, footprint, cross, surface, height, shadowed, stable, margin, contact_area]`. Rationale: the LOCKED phase goal explicitly lists shadowed as a metric that must have a single Rust implementation; success criterion #1 makes "no TS duplicate exists" mandatory for shadowed too; consensus depends on it. This is a faithful extension of D-05 covering a gap the original spec omitted.

Per-direction logic inside `score_all_directions`:
- Reconstruct mesh from positions/normals/areas via existing `reconstruct_mesh` helper.
- Compute hull via `decimate::sample_for_hull` + `hull::compute_hull` (once, outside the loop).
- For each direction `i` in 0..total:
  - Normalize dir.
  - If `refine_iters &gt; 0`: call `refine_once` with RNG seeded by `rng::seed_from_direction(&dir, 0)` (existing pattern). Else use the input dir.
  - `let c = scoring::score_components(&best_dir, &mesh, critical_angle_deg, 64)` — returns ScoreComponents (overhang, footprint, max_cross, surface_quality, height).
  - `let shadowed = scoring::shadowed_overhang_fraction(&best_dir, &mesh, critical_angle_deg, 32, 0.02)` — call 8 yaw samples per TS `minShadowedOverhang` lines 519-544 (rotate the perpendicular basis e1,e2 by theta = s*2π/8 for s in 1..8, take the min). Per assumption A7 (research line 805).
  - `let stab = stability::check_stability(&best_dir, &mesh, &hull)`.
  - Compute quaternion: `let q = yaw::full_quaternion(&best_dir, &mesh)`.
  - Push 13 floats: `q[0..4], c.overhang, c.footprint, c.max_cross, c.surface_quality, c.height, shadowed, stab.stable as f32, stab.margin, stab.contact_area`.
  - Progress callback: if `Some(cb) = progress` AND `i % 10 == 0`, call `cb.call2(&JsValue::UNDEFINED, &JsValue::from_f64(i as f64), &JsValue::from_f64(total as f64))`. Wrap the call in `let _ = ...` (ignore JS errors — research Pattern 2).
