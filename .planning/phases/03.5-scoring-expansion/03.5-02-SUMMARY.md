## Decisions Made

- **import: 'default' in glob:** Vite's `import.meta.glob('./*.json', { eager: true })` returns module namespace with 6 keys (default + each property). Using `import: 'default'` returns only the JSON object — clean 5-field profiles.
- **try/catch per direction in computeSlice:** per T-03.5-03 mitigation, if WASM refine throws for one direction, fall back to raw penalty + 0 variance + console.warn. Prevents one bad direction from killing all computation.
- **Seed=42 for Varita Mágica:** Fixed default seed for the manual refine button. Deterministic and reproducible.
